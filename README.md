## Project Overview
This project analyzes sales data in the United States to identify seasonal trends and sales performance across different cities and regions.

The dashboard provides insights into:
- Sales trends over time (seasonality)
- Sales distribution by region and state
- Top-selling products
- Overall sales performance indicators

## Tools Used
- Power BI
- Excel / CSV Dataset

## Dataset
The dataset contains sales records including:
- Order Date
- Sales Amount
- Product Name
- City, State, and Region

## Dashboard Highlights
- **Average Sales KPI**
- **Sales by Order Date** to analyze seasonality
- **Sales by Region and State**
- **Top Products by Sales**

## Key Insights
- The West region shows the highest total sales.
- California and New York are the top-performing states.
- Sales fluctuate across the year, indicating seasonal patterns.
- A small number of products contribute significantly to total sales.
